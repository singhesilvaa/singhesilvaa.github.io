About Me

I’m a Ph.D. student in Computer Science at the University at Albany, SUNY, advised by Prof. Jeong-Hyon Hwang. My research explores the design and optimization of on-disk and in-memory key-value stores — building scalable, high-performance data infrastructures for next-generation storage systems. My work bridges data management theory and systems engineering, with ongoing interests in storage architectures, indexing strategies, and real-time data processing frameworks.

Previously, I was a Senior Software Engineer at Sysco LABS, where I developed and optimized large-scale microservice architectures, data pipelines, and synchronization frameworks for enterprise applications. These experiences shaped my drive to build systems that are not only efficient but also resilient, observable, and impactful in real-world environments.

I’m passionate about data-intensive systems, storage research, and building tools that make data faster, smarter, and more accessible.

An Experimental Evaluation and Investigation of Group Farness Estimation Techniques

A comprehensive experimental study of group farness estimation methods in large-scale graphs. While numerous algorithms exist for single-source farness (closeness) estimation, group-level estimation remains underexplored despite its significance in applications such as community detection, influence maximization, and network optimization.

  • Reviewed and analyzed existing single-source farness estimation algorithms and their applicability to group-level estimation.
  • Extended sampling-based and cardinality estimation (CE) techniques—including KMV and Cohen’s HIP permutation estimator—for scalable group farness estimation.
  • Investigated trade-offs between accuracy, memory overhead, and runtime efficiency across various sampling strategies.
  • Conducted controlled experiments on extended group farness estimation techniques, where Cohen’s HIP-based estimator demonstrated modest yet consistent accuracy (≈2–5%) and scalability gains on large graphs.

This work offers quantitative insights into how sampling and cardinality estimation choices influence the accuracy and scalability of group farness estimation in large graph analytics.

Mimix Data Transformer

A modular Java library designed for efficient and extensible processing of Mimix event streams — enabling transformation, mapping, and post-processing workflows for relational database integration and streaming pipelines such as AWS Kinesis. The framework supports selective event filtering, JSON-based mapping configurations, and user-defined hooks for custom logic, all packaged as a Maven-compatible artifact for seamless inclusion in enterprise systems.

  • Implements fine-grained event filtering and JSON-based mapping for flexible and high-throughput data ingestion.
  • Supports custom transformation logic and post-processing hooks (e.g., logging, notifications, and audit pipelines) for workflow extensibility.
  • Integrates with AWS Kinesis for real-time event streaming and replication of Mimix event data to downstream systems.
  • Provides a pluggable architecture for extending filters, transformers, and database writers.
  • Distributed as a Maven-compatible Java library enabling rapid adoption and integration across data engineering environments.
  • Demonstrated via a short walkthrough video: Watch on YouTube.

Ideal for real-time replication of Mimix event streams, integration with streaming systems like AWS Kinesis, and the construction of scalable event-to-table data pipelines for modern data architectures.
